# Changelog

## [0.1.0] - 2026-02-01

### Added

- Core gameplay mechanics
  - Lemming spawning and movement
  - Falling with fatal distance calculation
  - Wall detection and turning
  
- Three abilities
  - Blocker: stops walking, turns other lemmings
  - Builder: places 12 diagonal stairs
  - Digger: digs straight down through terrain

- Terrain system
  - Pixel-perfect collision detection
  - Destructible dirt terrain
  - Indestructible steel terrain

- 5 tutorial levels
  - Just Walk: direct path (tutorial)
  - Turn Around: use blocker
  - Build a Bridge: use builder
  - Dig Down: use digger
  - The Gauntlet: all three abilities

- UI components
  - Title screen with level select
  - HUD with timer, saved count, progress bar
  - Ability panel with keyboard hints
  - Win/lose result screens

- Full keyboard controls
  - [1/2/3] Select abilities
  - [P] Pause/Resume
  - [R] Restart
  - [ESC] Cancel selection
